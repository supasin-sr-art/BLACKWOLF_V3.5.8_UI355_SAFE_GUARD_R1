Creator Access Profile Builder

สร้างไฟล์ access-profile.js แบบออฟไลน์ ระบบบันทึกเฉพาะ Salt + PBKDF2 Hash และไม่ใส่รหัสผ่านจริงไว้ในโค้ด

วิธีใช้: แตก ZIP → เปิดไฟล์นี้ → กำหนดรหัส → นำ access-profile.js ที่ดาวน์โหลดไปวางทับไฟล์เดิมในโฟลเดอร์ BLACKWOLF → ห้ามแนบรหัสผ่านจริงไปกับ ZIP ที่ส่งต่อ
คำแนะนำรหัสผ่าน: อย่างน้อย 12 ตัวอักษร มีตัวใหญ่ ตัวเล็ก ตัวเลข และสัญลักษณ์
